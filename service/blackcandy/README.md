# Blackcandy service

## Installation

- Create `.env` file with the following values
```bash
# Port number for the dashboard
PORT=

# Time zone (https://en.wikipedia.org/wiki/List_of_tz_database_time_zones)
TZ=

# Rails env value (can be production)
RAILS_ENV=

# Set to some long random string
SECRET_KEY_BASE=

# Path in the container to store all the music
MEDIA_PATH=
```
